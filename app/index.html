    Transect (depth profile)

    Draw a line across the seafloor to read its bathymetric profile — sampled in your browser from the same DEM as the map.

    Species occurrences (GBIF / OBIS)

    Plot georeferenced records for a taxon in the current view. Points become stations you can style and export.

    Cartography assistant optional

    An optional helper that answers figure-design questions grounded in the selected journal's spec. It's off by default and never needed — the tool works fully without it. It uses your own Anthropic API key (stored only in this browser) and calls the API directly.

    API key & model

    Get a key at console.anthropic.com. Usage is billed to your key. The key never leaves your browser except in the direct call to Anthropic.

    Cartographic furniture

    The data-attribution line is always included — it keeps your figure compliant with the data licences.
